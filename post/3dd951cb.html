QQBot用户使用手册V1.0

发表于2025-01-23|更新于2025-01-23
|总字数:1013|阅读时长:3分钟|浏览量:

基础说明

​ - By BR

​ 本项目基于NapCat+python3.10，是在QQBot基础上的二次开发，存在个人魔改部分，可在Window及Linux上部署

​ @信息，命令信息，文本信息等，每个信息之间是以一个空格分隔的

​ 普通用户权限为1，管理员权限为2，超级管理员权限为3

​ 作用范围：friend(仅私聊)，group(仅群聊)，public(私聊与群聊均可)

命令目录

/help

  • 来源：系统命令

  • 作用范围：public

  • 权限要求：1

  • 用法：/help

  • 作用：查看当前环境下的命令目录

/reload

  • 来源：系统命令

  • 作用范围：friend

  • 权限要求：3

  • 用法：/reload

  • 作用：重载所有插件

/点赞

  • 来源：QQ_like插件

  • 作用范围：friend

  • 权限要求：2

  • 用法：/点赞 [subcommand] [arguments]

  • 参数：

    subcommand:
    set <status>          设置插件状态，"True,true,1"表示启用，"False,fasle,0"表示禁用
    add <QQ_id>           添加待点赞QQ号
    del <QQ_id>           删除待点赞QQ号

  • 作用：明天晚上12点整为列表用户的名片点赞

/比赛监听

  • 来源：match_monitoring插件

  • 作用范围：friend

  • 权限要求：2

  • 用法：/比赛监听 [subcommand] [arguments]

  • 参数：

    subcommand:
    set <status>          设置插件状态，"True,true,1"表示启用，"False,fasle,0"表示禁用
    add <QQ_group_id>     添加通报群聊QQ号
    del <QQ_group_id>     删除通报群聊QQ号

  • 作用：监听GZCTF平台比赛实况，包括一血二血三血，题目上新，新增公告等的提示

/查询比赛

  • 来源：get_match插件

  • 作用范围：public

  • 权限要求：1

  • 用法：/查询比赛 [arguments]

  • 参数：

    arguments					表示指定页码，默认为1

  • 作用：查询近期国内网安类比赛，消息来源：ProbiusOfficial/Hello-CTFtime: 【Hello CTF】国内外赛事聚合 (github.com)

/查询国外比赛

  • 来源：get_match插件

  • 作用范围：public

  • 权限要求：1

  • 用法：/查询国外比赛 [arguments]

  • 参数：

    arguments					表示指定页码，默认为1

  • 作用：查询近期国内网安类比赛，消息来源：ProbiusOfficial/Hello-CTFtime: 【Hello CTF】国内外赛事聚合 (github.com)

/响应群聊

  • 来源：just_group插件

  • 作用范围：friend

  • 权限要求：2

  • 用法：/响应群聊 [subcommand] [arguments]

  • 参数：

    subcommand:
    set <status>          设置插件状态，"True,true,1"表示启用，"False,fasle,0"表示禁用
    add <QQ_group_id>     添加允许响应群聊
    del <QQ_group_id>     删除允许响应群聊

  • 作用：限制Bot能够响应的群聊，在允许响应群聊外的群聊，Bot将会无视其信息

/ai

  • 来源：tongyi插件

  • 作用范围：public

  • 权限要求：1

  • 用法：/ai [subcommand]

  • 参数：

    subcommand:
    cls          		清除发出信息的用户与ai的聊天记录

  • 作用：用户层对于ai的管理

/ai管理

  • 来源：tongyi插件

  • 作用范围：friend

  • 权限要求：2

  • 用法：/ai管理 [subcommand] [arguments]

  • 参数：

    subcommand:
    set <status>          		设置插件状态，"True,true,1"表示启用，"False,fasle,0"表示禁用
    add <QQ_id>     	  		添加黑名单用户
    del <QQ_id>     	  		删除黑名单用户
    cls [QQ_id]			  		清除指定用户与ai的聊天记录(保留预设),未传入QQ_id参数则全清除
    ccls [QQ_id]		  		清除指定用户与ai的聊天记录(不保留预设),未传入QQ_id参数则全清除
    preset [set <preset_id>]	设置当前预设，若未传入set命令，则展示当前使用预设及可选预设
    show						将当前存在会话的对话信息保存并以md文档发送给发出命令者

  • 作用：管理层对于ai的管理

/关于

  • 来源：about插件

  • 作用范围：public

  • 权限要求：1

  • 用法：/关于

  • 作用：展示Bot基本信息

/夸夸

  • 来源：praise插件

  • 作用范围：public

  • 权限要求：1

  • 用法：/夸夸 [argumnets]

  • 参数：

    arguments					表示对谁使用夸夸，如果不存在该参数，默认对使用命令者使用夸夸

  • 作用：从词库中随机抽取夸奖文案并回复
